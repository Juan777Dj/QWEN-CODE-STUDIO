# 🚀 QwenCode Studio

**Tu IDE potenciado con Qwen 3.7**

QwenCode Studio es un entorno de desarrollo moderno y ligero que combina la potencia de Monaco Editor (el mismo motor de VS Code) con capacidades avanzadas de IA a través de Qwen 3.7.

## ✨ Características Principales

- 🎨 **Editor Monaco** - El mismo motor de VS Code con syntax highlighting para 100+ lenguajes
- 🤖 **Asistente IA Integrado** - Chat con Qwen 3.7 para ayudarte a programar
- 📁 **Explorador de Archivos** - Navegación intuitiva por tu proyecto
- 🎯 **Optimización de Tokens** - Gestión inteligente del presupuesto de tokens
- 🌓 **Tema Oscuro** - Diseñado para largas sesiones de coding
- ⚡ **Rápido y Ligero** - Basado en Vite + React + TypeScript

## 🛠️ Stack Tecnológico

- **Frontend:** React 18 + TypeScript
- **Editor:** Monaco Editor
- **Build Tool:** Vite 6
- **Estilos:** CSS Variables
- **IA:** Qwen 3.7 (integración lista)

## 🚀 Inicio Rápido

```bash
# Instalar dependencias
npm install

# Iniciar servidor de desarrollo
npm run dev

# Construir para producción
npm run build
```
